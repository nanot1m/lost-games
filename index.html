Справка

Краткая памятка по управлению и текущим правилам Lost Games TD.

Как играть

  • Монстры идут справа налево и пытаются пробить защитный полукруг.
  • Волны обычных врагов дают 1 карту усиления для звена пушки.
  • После убийства босса выпадает 1 из 3 редких бафов линзы и открывается новый слот пушки.
  • Максимум можно открыть 5 пушек. Бафы звеньев можно вешать и на неустановленные, но уже открытые пушки.
  • Каждая установленная пушка получает только часть общей энергии двигателя. Чем больше активных пушек, тем слабее поток на каждую.

Управление

  • ЛКМ по пушке: выбрать пушку.
  • Shift + ЛКМ: мультивыбор пушек.
  • Drag мышью по выбранной пушке: перемещение по дуге стены.
  • Tab / Shift + Tab: переключение между установленными пушками.
  • W / S: плавно двигать выбранную пушку по стене.
  • 1 / 2 / 3: сменить профиль атаки выбранной пушки.
  • В 3D режиме drag по пустому полю вращает камеру, колесо мыши меняет зум.

Профили атаки

  • Круг: универсальная зона вокруг пушки.
  • Снайпер: редкие выстрелы по треугольной дальней зоне.
  • Дробовик: ближняя широкая зона с несколькими пулями.

Бафы

  • Обычные карты усиливают без штрафов.
  • Редкие и легендарные карты сильнее, но могут ослаблять другие характеристики.
  • Линза даёт глобальные эффекты всем установленным пушкам и не имеет негативных эффектов.
  • Элементальные эффекты конфликтуют: огонь снимает лёд, лёд тушит огонь.
